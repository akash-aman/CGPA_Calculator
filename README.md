## How to run 

- Installing pnpm 
```
    npm install -g pnpm
```
- Installing pnpm dependencies
```
    pnpm install 
```
- Run in Development Mode  
```
    pnpm run dev
```
- Run in Production Mode 
```
    pnpm run build
    pnpm run start
```
